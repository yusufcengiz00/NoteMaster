# NoteMaster

Kullanıcılar bu uygulamayı kullanarak not alabilir, notlarını depolayabilir ve istedikleri zaman tekrar görüntüleyebilirler. Kullanıcıların uygulamayı kullanabilmeleri için giriş yapmaları zorunludur.
